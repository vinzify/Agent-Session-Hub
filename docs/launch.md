## Reddit / Forum Intro

```text
I kept losing context across Codex CLI, Claude Code, and OpenCode, so I built a shared session browser for all three.

Agent Session Hub gives them one `fzf` picker with previews, aliases, and resume-in-project behavior. The generic entrypoint is `sessionhub`, and the direct daily commands are `csx`, `clx`, and `opx`.

It is a native Rust CLI, installs with one command, and works across bash, zsh, fish, PowerShell, and Windows cmd.
```
